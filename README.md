# ClimateGuard – Weather Risk Analysis

ClimateGuard is a frontend-focused project that aims to visualize environmental risks such as flooding, wildfire probability, and terrain elevation using geospatial data.

This repository currently contains the **static frontend prototype** of the project.

## Project Status

This version includes only the **HTML and CSS implementation** of the interface.
The **JavaScript functionality and API integrations will be implemented in later stages** of the project.

## Features Implemented

* Modern responsive dashboard UI
* Climate statistics cards
* Search interface for location-based analysis
* Risk results panel
* Map container layout (placeholder)
* Dark themed dashboard design

## Technologies Used

* HTML5
* CSS3
* Font Awesome Icons
* Google Fonts (Plus Jakarta Sans)

## Future Development

The following features will be added in upcoming updates:

* Interactive map using Leaflet.js
* Real-time location search
* Climate risk analysis logic
* API integrations for environmental data
* Dynamic risk visualization
* Downloadable analysis reports

## Note

This repository currently represents **only the static frontend component** of the ClimateGuard project.
The **JavaScript logic and dynamic functionality will be developed in future phases.**
